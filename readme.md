# 📚 Bookmark Manager

> Удобное десктопное приложение для управления закладками с вкладками, поиском, импортом/экспортом и авто‑закрытием.

![Screenshot](https://via.placeholder.com/800x400?text=Bookmark+Manager+Preview)

---

## ✨ Возможности

- **Вкладки** – группируйте закладки по темам (Графика, Код, Нейронки…).  
- **Карточки** – каждая закладка содержит название, описание, URL и картинку (по ссылке или загрузка).  
- **Drag & Drop** – перетаскивайте вкладки и карточки для изменения порядка.  
- **Поиск** – глобальный поиск по всем закладкам (название, описание, URL) с подсветкой совпадений и выпадающим списком.  
- **Настройка сетки** – выбирайте количество колонок (1–4), значение сохраняется в `localStorage`.  
- **Импорт из Chrome** – автоматически находит файл закладок Chrome (включая профили) и создаёт вкладку «Из Chrome».  
- **Экспорт в Chrome** – генерирует HTML‑файл формата Netscape Bookmark для импорта в Chrome.  
  - можно разбивать по папкам (по вкладкам) или экспортировать все закладки плоско.  
  - имя закладки = название + описание (если есть).  

- **Авто‑закрытие** – приложение завершается, когда закрыта вкладка браузера с ним.  
- **Адаптивный интерфейс** – тёмная тема, плавные анимации, кастомный скролл.

## 🚀 Запуск
Запустить exe файл из репозитория или<br>
 
Скачайте исходный код
```bash
git clone https://github.com/K-A-R-P/bookmark_manager.git
```
Установите зависимости из requirements.txt или командой
```bash
pip install Flask==2.3.3 flask-cors==4.0.0 flask-socketio==5.3.6 python-socketio==5.11.0 simple-websocket==1.0.0
```
Перейдите в папку с проектом запустите `app.py`
```bash
python app.py
```
После запуска браузер автоматически откроется по адресу `http://localhost:5000`.  
При желании можно собрать приложение в исполняемый файл с помощью PyInstaller
```bash
pip install pyinstaller
```
```bash
pyinstaller --clean --onefile --noconsole --add-data "templates;templates" --add-data "static;static" --icon=icon.ico --name="BookmarkManager" --hidden-import=engineio.async_drivers.threading --hidden-import=simple_websocket app.py
```
База данных создаётся рядом с приложением

## 📖 Использование

### Управление вкладками
- **Создать** – кнопка **«Новая вкладка»**.
- **Переименовать** – двойной клик по названию вкладки.
- **Удалить** – включите режим редактирования (кнопка с карандашом), затем нажмите крестик на вкладке.
- **Перетащить** – зажмите вкладку и переместите в нужное место.

### Управление карточками
- **Добавить** – кнопка **«Добавить»** → заполните поля (название, описание, URL, картинка).
- **Редактировать** – клик по карточке (кроме кнопки «Перейти» и корзины).
- **Удалить** – наведите на карточку, нажмите корзину.
- **Перетащить** – зажмите карточку и переместите внутри той же вкладки (меняется порядок) или в другую вкладку.

### Поиск
- Введите запрос в поле поиска – через 300 мс появится выпадающий список с первыми 10 результатами.
- Нажмите **Enter** – откроется модальное окно со всеми результатами.
- Результаты меняются с каждым введенным символом

### Импорт из Chrome
- Нажмите кнопку **«Импорт из Chrome»**.  
- Приложение автоматически найдёт файл `Bookmarks` (или `Bookmarks.bak`) в профилях Chrome и добавит все закладки в новую вкладку «Из Chrome».

### Экспорт в Chrome
- Нажмите кнопку **«↓ Chrome»**.  
- В появившемся окне выберите, разбивать ли закладки по папкам (вкладки → папки) или нет.
- Нажмите **«Скачать HTML»** – файл `bookmarks-chrome.html` будет сохранён.  
- Импортируйте его в Chrome через **Закладки → Импорт закладок и настроек → Файл HTML**.

## 🛠 Технологии

- **Backend**: Python 3, Flask, Flask‑SocketIO, SQLite.
- **Frontend**: HTML, CSS (Tailwind), JavaScript, FontAwesome.
- **Прочее**: WebSocket для отслеживания открытых вкладок (авто‑закрытие).

## 🗄️ База данных

- Таблица `tabs`: `id`, `name`, `position`
- Таблица `cards`: `id`, `tab_id`, `title`, `description`, `url`, `image`, `position`

При первом запуске создаётся одна вкладка с именем **«Вкладка»**.
